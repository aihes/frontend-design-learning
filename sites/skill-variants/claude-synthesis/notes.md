# Claude Synthesis Variant

## Design Direction
Dark minimalism with micro-interactions and progressive disclosure. Evaluates Claude's native design capability without external skill guidance.

## Design Decisions
- **Color**: Dark background (#0a0a0b) with mint-green accent (#6ee7b7) and warm gold secondary (#fbbf24) — high contrast, modern tech aesthetic
- **Typography**: Inter/system sans-serif, monospace labels for categorical info — clean hierarchy without decorative fonts
- **Layout**: Single-column hero with left-aligned text, full-width stats bar, bordered project cards with 4-column evidence grid
- **Interaction**: Hover state transitions on cards (border glow, subtle lift), pulsing nav dot, smooth scroll, selection color matching accent
- **Structure**: Sticky nav with backdrop blur, modular sections (fit/cases/background/stack), pill-shaped CTA buttons

## Differentiators from Other Variants
- Only dark theme variant among 7 versions
- Only variant with CSS transitions/hover effects
- Uses backdrop-filter blur on navigation
- Gradient text effect on hero heading keyword
- Monospace labels for section headers and evidence categories
- Stat cards with hover background transition
- Rounded pill buttons instead of rectangular/squared
- No decorative elements (no serif fonts, no rotated cards, no poster layouts, no grid-paper backgrounds)
