Resume Site Design Tool Benchmark

把同一个脱敏简历站 brief 丢给 Variant、Figma Make、v0、Lovable、Bolt、Relume 等工具，测试哪些能真正生成设计，哪些更适合作为参考库。

结论

真正跑出结果

Relume 可以未登录生成 sitemap 和 wireframe，但会编造项目、推荐语和人物信息，只能用结构，不能信内容。

能输入但被登录挡住

Variant、Figma Make、v0、Lovable、Bolt 都能进入 prompt 流程，但提交后需要登录或私有会话。

最稳定的参考来源

Godly、Land-book、Saaspo、Awwwards、Mobbin、Page Flows、Refero 更适合做视觉和流程参考。
