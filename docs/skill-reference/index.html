Frontend Design Skill Reference

本页是可公开分享的 Skill 参考索引。Markdown 文件保留更完整的输入、输出、工作流和验收标准；这里提供快速入口。

公开原则：不包含本机绝对路径、公司名、内部主机、内部项目和账号信息；不复制本机 Skill 原文，只保留可复现的行为说明。

阶段参考文件用途
叙事重组interactive-portfolio把简历或作品集从时间顺序改成证明顺序。
视觉方向frontend-design选择明确页面气质，避免通用 AI 模板。
多方案探索design-shotgun方向不确定时先生成多套差异化方案。
审美校准design-taste-frontend清理紫蓝渐变、三列卡片、默认字体等 AI 味。
设计系统design-consultation沉淀品牌、字体、颜色、间距、动效和设计原则。
Tailwind 落地tailwind-design-system把设计方向变成 token 和组件 primitive。
布局稳定shadcn-layouts处理 app shell、高度链、滚动和 grid/flex 问题。
CSS 细节frontend-css-patterns提供字体、色彩、动效、背景和空间组合模式。
转化页landing-page-design设计 landing page 的首屏、CTA 和信任证据。
发布审查claude-page-review用第二模型视角检查文案、层级、真实性、隐私和移动端。

怎么用这些文件

把对应 Markdown 作为 prompt 附件或复制给 Codex，让它按该工作流执行。它们不要求安装同名 Skill。
