Case 01

智食 AI

AI 饮食推荐小程序 · 独立开发 · 2026.05 至今

AI 决策 云开发 地图服务 LLM API
User Problem

外出就餐的痛点不是信息不足而是选择过载，用户需要的是决策辅助而非更多选项。

Product Call

把输入收敛为 4 个关键维度，规划 6 大推荐场景，结果以菜品、真实餐厅和情绪化文案呈现。

Execution

接入多个地图服务 API，通过 Prompt 约束 AI 只从真实店铺池选店，确保推荐可落地。

Signal

1 周完成产品设计、开发、安全加固与发布，8 个云函数模块支持多菜系和海外检索。

Case 02

GameHub

游戏工具平台 · 独立开发 · 2026.04 至今

Next.js Supabase UGC 多语言
User Problem

跨服玩家缺少可检索、可分享的专业工具，分散在社区的经验难以结构化复用。

Product Call

设计分级体系、多维标签和组合筛选，把分散经验变成结构化工具。

Execution

实现双登录、UGC 上传、三语切换；CDN 管理 60+ 资源图片，SEO 全自动生成。

Signal

5 天上线双端，性能评分满分；首月 2,800+ 独立访客、3,500+ 浏览、200+ UGC。

Case 03

EthiScore

企业伦理评分应用 · 团队成员 · 2025.09 - 2025.12

ESG 评分模型 商业模式 Figma
User Problem

消费者和投资者很难快速判断企业伦理表现，缺乏可解释、可比较的评分工具。

Product Call

补充 20+ 评估指标，将四大维度落实为可操作的评分框架。

Execution

完成用户细分、订阅定价和商业模式画布，梳理三款竞品确定差异化定位。

Signal

搭建 4 页 Demo，接入后端和 AI API，以真实企业数据完成评分验证。
